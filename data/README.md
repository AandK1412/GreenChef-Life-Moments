# Data

## No real subscriber data is used in this repository

The capstone Statement of Work (Appendix H) requires that **structurally
representative dummy datasets** be used for methodology design, code development,
and all public-facing materials, with real HelloFresh-provided client data
confined to internal analysis under NDA.

This repository contains **only synthetic data**. Nothing here reproduces or
derives from real subscriber records.

## `synthetic_cancellations.csv`

Generated by `scripts/generate_data.py` (logic in
`src/life_moments/synthetic.py`). The latent class mix and life-context weights
are tuned to the report's Q1 2026 headline figures (Appendix E):

| | Report | Synthetic (this repo, seed 5905) |
|---|---|---|
| Total records | 14,231 | 14,231 |
| High-Potential | 21.86% | ~21.8% |
| Low-Potential | 68.02% | ~67.8% |
| Release | ~10.1% | ~10.4% |
| Urgent priority | ~13.0% / 1,848 | ~13.3% / ~1,900 |
| Weekly High-Potential range | 334–404 (mean 389) | ~369–408 (mean ~387) |
| Dominant life-context | Budget Pressure | Budget Pressure |

Columns:

| Column | Description |
|---|---|
| `order_id` | Synthetic identifier (`GC-1xxxxx`) |
| `week` | Q1 2026 week number, 1–8 |
| `comment` | Synthetic cancellation comment text |

## Regenerate

```bash
python scripts/generate_data.py                 # 14,231 records
python scripts/generate_data.py --rows 2000     # smaller sample
```
